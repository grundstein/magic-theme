@grundstein/magic-theme

this is the @magic-themes grundstein theme. It is being used as the foundational theme for all @grundstein pages.

installation

npm install --save --save-exact @grundstein/magic-theme

usage

require

// config.mjs export default {  // ... other config   THEME: '@grundstein/magic-theme',}

theme vars

colors

this theme exports and uses the following colors.

  • vars.white: #ffffff
  • vars.neutral: #5a5a5a
  • background
    • vars.background.light: #e0e0e0
    • vars.background.dark: #212121
  • link
    • vars.link.light: #212121
    • vars.link.dark: #c4c4c4
    • link.hover
      • vars.link.hover.light: #f09d09
      • vars.link.hover.dark: #f09d09
  • primary
    • vars.primary.neutral: #f09f09
    • vars.primary.light: #ffee88
    • vars.primary.dark: #ffa000
  • secondary
    • vars.secondary.neutral: #5a5a5a
    • vars.secondary.light: #d8d8d8
    • vars.secondary.dark: #0a0a0a
  • text
    • vars.text.light: #212121
    • vars.text.dark: #c4c4c4

max page width

vars.maxWidth: 1000px

fade duration:

vars.fadeDuration should be used for all css animations

vars.fadeDuration: 500ms

breakpoints

the following width breakpoints exist

  • vars.widths.tablet: 500px
  • vars.widths.laptop: 1024px
  • vars.widths.desktop: 1600px
